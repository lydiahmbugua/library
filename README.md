# Library Application

A responsive web application designed to manage a personal book collection. Users can add new books, track their reading status (Read/Unread), and remove books from their library.
This project was completed as part of The Odin Project curriculum.

## 📋 Features
- **Book Management:** Add new books with title, author, number of pages, and read status via a modal form.
- **Dynamic Updates:** Instant updates to the library list without page reloads.
- **Responsive Design:** Optimized layout for mobile, tablet, and desktop viewing.

## 🛠 Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Build Tools:** Git
- **Design:** Vanilla CSS
